# Privacy Policy for Karpathy Tweet Formatter

**Last Updated:** January 30, 2026

## Overview

Karpathy Tweet Formatter is a browser extension that formats tweets from @karpathy to make them easier to read. This privacy policy explains how the extension handles your data.

## Data Collection

**We do not collect any data.** This extension:

- Does NOT collect any personal information
- Does NOT track your browsing history
- Does NOT send any data to external servers
- Does NOT use cookies or any tracking mechanisms
- Does NOT require user accounts or authentication

## How the Extension Works

The extension operates entirely within your browser:

1. It runs only on x.com (Twitter) pages
2. It looks for tweets authored by @karpathy
3. It adds visual line breaks after sentences to improve readability
4. All processing happens locally in your browser

## Permissions Explained

The extension requires the following permissions:

- **Host permissions for x.com and twitter.com**: Required to run the content script that formats tweets. The extension only activates on Twitter/X pages and cannot access any other websites.

## Data Storage

This extension does not store any data locally or remotely. It has no persistent storage, no local databases, and no cloud connections.

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last Updated" date above.

## Contact

If you have questions about this privacy policy, you can contact us by opening an issue on the extension's GitHub repository.

## Open Source

This extension is open source. You can review the complete source code to verify these privacy claims.
